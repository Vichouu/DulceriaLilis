⚙️ PASOS PARA CONFIGURAR EL PROYECTO
------------------------------------
1️⃣  Clonar el repositorio local con Git:
     git clone [URL-del-repositorio]

2️⃣  Instalar y configurar WAMP Server.

3️⃣  Crear la base de datos:
     Nombre: Dulcerialilis  
     Herramienta: PHPMyAdmin

4️⃣  Importar la última versión del archivo .sql del proyecto.

5️⃣  Crear el usuario administrador de la base de datos (si no fue creado por el script .sql):
     - Nombre: Admin
     - Base de datos: Dulcerialilis

6️⃣  Otorgar todos los privilegios para la base de datos:
     - En PHPMyAdmin, marcar “Otorgar todos los privilegios”.
     - Aplicar los privilegios al usuario admin de la BD Dulcerialilis.
